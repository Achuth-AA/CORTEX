# 🗂️ Projects Workspace

A collection of six self-contained full-stack apps. Each lives in its own folder
with its own README, tech stack and run instructions.

| Project | Folder | What it is | Stack |
|---------|--------|-----------|-------|
| **🧩 Foliofy** | [`PortfolioGenerator/`](PortfolioGenerator) | Fill a form → download a deploy-ready portfolio website ZIP (site + résumé + QR + deploy kit) | React · Flask · SQLite |
| **📝 SyncScribe** | [`DocsClone/`](DocsClone) | Real-time collaborative document editor with templates & auto-save | React · Quill · Flask-SocketIO · SQLite |
| **∿ Algoscope** | [`AlgoVisualizer/`](AlgoVisualizer) | Visualize 8 algorithms + code them in-browser and run against tests/edge cases | React · Flask · SQLite |
| **📄 ResumeIQ** | [`AI-Resume-Analyzer/`](AI-Resume-Analyzer) | AI résumé analyzer — scores, ATS keywords, suggestions | React · Flask · **Google Gemini** · SQLite |
| **🔒 Lumina Chat** | [`ChatbotusingSockets/`](ChatbotusingSockets) | Real-time end-to-end (AES-256) encrypted chat with rooms & DMs | React · Tailwind · **Node/Express** · Socket.io · SQLite |
| **🛍️ Velora** | [`ECommerce/`](ECommerce) | Animated e-commerce storefront with a 3D product viewer & checkout | React · Tailwind · Three.js (no backend) |

Each folder's `README.md` has the full feature list, tech stack, project
structure, and step-by-step run instructions.
